Webpages, Documents and Structures

Stage-1: Make your Web Page:

The Basics of Web and HTML

  • Web is a medium which enables communication among computers.
  • Process of communication involves, A User(computer) makes a request---> Internet (transmits) --->Servers.
  • Internet uses HTTP to communicate from User to Server and server responds back to user.
  • Plain text, HTML, Images, videos are few examples of files we find on the web.
  • HTML stands for Hyper Text Markup Language, Hyper text - what we see and Markup Language means - how we can make it look.
  • World Wide Web is a collection of HTML documents.

    • HTML Tag and HTML Element

  • The most common terms we come across while coding are TAGS and Element.
  • HTML tag is always contained in angled brackets.
  • As a general rule Element is composed of an Opening tag, Content and a Closing tag. However it does not limit itself to the said rule.
  • HTML Attributes such as Anchor tags help us link the documents with other websites in reference to.
  • Image Tag enables us to attach an image to our documents. Scr and Alt are its attributes.

    • Inline Vs Block Elements

  • To add line spaces among text we can use break tag (br tag). b,em,br,img,a tags are some of the examples of Inline elements.
  • Block Elements are tags that create an invisible box arounfd the text example: p tag.
  • Elements with only opening tag is termed as Void Tag

    • Computers are stupid means they interpret exactly as per instructions. Even a slight mistake can lead to huge problems.

    Stage-3: Adding CSS Style to HTML Structure:

    Cascading Style Sheets

  • CSS code controls the Style of HTML elements.
  • It is a great tool which avoid repitition by adding a style to a class attribute.
  • It enables seperation of document content from document presentation by including elements such as layout, fonts and colors.
  • Desendent elements inherit only text related properties.

    • The Box Model, Box-Sizing and Box-Positioning

  • HTML elements are boxes and each box has 4 components.
  • It can often be hard to get right size of a box.
  • Size in percentage and broder-box are the two techniques to deal sizing issues.
  • Different browsers display same code differently.
  • Divs are block elements, they take up width of entire page.
  • Adding rule display:flex; lets divs appear next to each other.
